# 🎀 TOOQ - متجر إلكتروني للملابس والإكسسوارات النسائية

## 📝 نبذة عن المتجر
**TOOQ طوق** - موقع تسوق إلكتروني عصري واحترافي متخصص في عرض وبيع الإكسسوارات والملابس النسائية.

**الرسالة:** الأناقة بسعر مناسب

**الفئة المستهدفة:** نساء 20-30 سنة وجميع السيدات

---

## 🛠️ التقنيات المستخدمة

### Frontend (الموقع)
- React.js
- Tailwind CSS
- Redux
- Axios

### Backend (الخادم)
- Node.js + Express
- MongoDB
- JWT Authentication
- Multer (رفع الصور)

### التطبيق
- Flutter Flow (iOS + Android)
- Firebase

### خدمات خارجية
- Sham Cash (الدفع)
- WhatsApp API (التواصل)
- Google Analytics

## 🚀 البدء السريع

### المتطلبات
- Node.js v16+
- MongoDB
- npm أو yarn

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/abdullahadi4m-ai/tooq-store.git
cd tooq-store

# تثبيت الـ Frontend
cd frontend
npm install
npm start

# تثبيت الـ Backend (في terminal منفصل)
cd ../backend
npm install
npm start
```
